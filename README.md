# springboot-vue-run
## included dependencies
* vue 2
* vuetify
* vue-router
* springboot
* springboot-devtools
* springboot-web
* springboot-security
* springboot-jpa

## how to run? (for development)
* server
```shell
$ cd client && yarn serve
```
* client
```shell
$ cd server && ./gradlew bootRun
```
